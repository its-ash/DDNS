ROUTER CONFIGURATION

⚙️ DDNS Update URL

Configure your router with the following settings:

  • Service: Custom or DynDNS Compatible
  • Server: {{ base_domain }}
  • Update URL: http://{{ base_domain }}/nic/update?hostname=%HOSTNAME%&myip=%IP%
  • Username: (from table above)
  • Password: (from table above)

Note: Some routers auto-detect IP, so myip parameter is optional.

💻 Manual Update via cURL

Test DDNS update manually:

curl -u USERNAME:PASSWORD "http://{{ base_domain }}/nic/update?myip=1.2.3.4"

🔗 Access via Hostname

To redirect to router IP when accessing hostname:

http://{{ base_domain }}/redirect

Set Host header to your hostname, or configure DNS to point to this server.
